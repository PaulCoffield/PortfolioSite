Semantic UI

In the process of using Semantic UI to create an interface, I found a UI Kit for Adobe XD - which was great, until XD started being a little less stable than I'd have liked. I found a few similar options that were available for Sketch but most of these were not as complete as the XD equivalent.

This project is my attempt to rectify this!

I find that things are often learnt best when you learn the theory, then put it into practice. So I am using this project as a vehicle for learning more about creating a UI Library in Sketch - and all that involves, including:

Creating symbols:

  • Best practice approach/es
  • Overrides
  • Symbol inception

Version control with Sketch:

  • GitHub
  • Zeplin

Giving back:

  • Once complete, I plan to make the Sketch library available to anyone else in my position that would like to make use of it to visual Semantic UI based interfaces in Sketch.
